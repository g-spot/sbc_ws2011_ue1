Getting the Source

The source for an OpenJMS release can be obtained from the Downloads page.

Instructions for obtaining the latest source from the CVS repository can be found on the CVS page.

More detailed instructions on accessing the source can be found on the CVS repository page.

If you intend to contribute changes to OpenJMS, it is recommended that you use the latest source from the CVS repository.
